Description: Nous avons créé une API qui resence les commentaires, les notes et les durée des séjours dans un Pays. Ces données ont étaient saisie par nos utilisateurs. Création de Oihan Marraud et Romain VACHÉ


Lien général: http://iparla.iutbayonne.univ-pau.fr/~rvache001/S4/projet/apiNotesPays.php/

Ce lien vous permet d'acceder à toutes les données saisies par nos utilisateurs.


Vous pouvez trier les données par pays, note ou bien durée de séjour. Exemple de lien :

http://iparla.iutbayonne.univ-pau.fr/~rvache001/S4/projet/apiNotesPays.php/?pays=France

http://iparla.iutbayonne.univ-pau.fr/~rvache001/S4/projet/apiNotesPays.php/?note=5

http://iparla.iutbayonne.univ-pau.fr/~rvache001/S4/projet/apiNotesPays.php/?tempsSejour=10


Voici, pour compléter la documentation, un exemple de code javascript fetch qui utilise notre API :

fetch(http://iparla.iutbayonne.univ-pau.fr/~rvache001/S4/projet/apiNotesPays.php/?pays=Australia, {
"method": "GET",
"headers": {
}
})
.then(response => {
console.log(response);
return response.json()
})
.catch(err => {
console.error(err);
});
